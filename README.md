#### Install script on Termux
``` bash
$ pkg update && upgrade  
$ pkg install python  
$ pip install bs4  
$ pip install requests  
$ pip install mechanize  
$ pip install futures
$ pkg install git 
$ rm -rf yulia
$ git clone https://github.com/Yuliaulfa/yulia
```
#### Run script
``` bash
$ cd yulia
$ python yulia.py  
```
